Fall 2021: Beginner Python Programming


This course covers the fundamentals of programming in Python. Students will learn about the applications of programming in Python and become familiar with the logic used in programming, including if/else statements, loops, and functions. Python containers like lists and dictionaries will also be covered. To help solidify these concepts, students will build several minigames throughout the course. At the end of the course, students will use their new programming knowledge to create a game called Bear in a Forest, where the player moved a bear around a grid to catch a deer.
No prior knowledge is required for this course, and everybody is welcome. In order to be able to complete the course, students will need an internet connection, a device capable of running Zoom, and a computer that can access https://replit.com/. This course was taught online by the OpenBrackets teachers.


This class was 3 times a week on Mondays, Wednesdays, and Fridays, from 5pm to 6:30pm Pacific Time. Classes started on Monday, October 4th and ended on Friday, November 12th.
