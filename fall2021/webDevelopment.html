Fall 2021: Beginner Web Development


This course covers the fundamentals of building a website with HTML and CSS. Students will learn about webpage structure and HTML tags. We will also teach CSS selectors, style properties, and the box model. Throughout the course, students will create and host websites on replit. Note that since this is an introductory course, JavaScript will not be covered.
No prior knowledge is required for this course, and everybody is welcome. In order to be able to complete the course, students will need an internet connection, a device capable of running Zoom, and a computer that can access https://replit.com/. This course was taught online by the OpenBrackets teachers.


This class was 2 times a week on Tuesdays and Thursdays, from 5pm to 6:30pm Pacific Time. Classes started on Tuesday, October 5th and ended on Thursday, November 11th.
